<h1 align="center" style="font-weight: bold;">Candy</h1>

## About

Candy is a platform for order management, developed with Next.js and React.

With Candy, you can register, view, and edit orders and their products.

This project was created to meet the needs of a small business.

## How to Use

1. To run this project, you need to have the API running. You can find it <a href="https://github.com/MarcsLeao/candy-website-api" target="_blank">here</a>.

2. Run the following commands:

```bash
npm install
```

```bash
npm run dev
```

3. Open the following URL in your browser: http://localhost:3000.
